List of local URLs:

All these URLs start with local:// and are local web files of the proxy. Type the URLs in the search bar above.

• local://apps - Opens all the apps.

• local://cheats - Opens all the cheats and hacks.

• local://credits - Credits page.

• local://games - Opens all the games.

• local://home - The home or the new tab page.

• local://list - This page.

• local://review - Write a review page.

• local://settings - Settings page.

• local://updates - Updates page or the changelog.

• local://welcome - The welcome page, or the first page you will see when entering the proxy. This page automatically redirects you to the home page if you were on it within the last 30 days.
